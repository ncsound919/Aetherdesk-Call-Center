# AetherDesk Agent UI

This is a premium React-based dashboard for agents to manage incoming AI-orchestrated calls.

## Quick Start

1. Ensure the backend is running on port 8001.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000` (or the port Vite selects).

## Key Features
- **Real-Time Queue Monitoring**: Automatically polls the backend for new calls.
- **Glassmorphism UI**: A sleek, modern dark mode design system.
- **One-Click Claiming**: Seamlessly transition from AI-led to Agent-led calls.
- **Live Preview**: See transcriptions as they happen.
